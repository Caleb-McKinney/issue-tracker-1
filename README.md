# Issue Tracker

This is the boilerplate for the Issue Tracker project. Instructions for building your project can be found at https://www.freecodecamp.org/learn/quality-assurance/quality-assurance-projects/issue-tracker

Deployment note:
- To expose FCC test results (e.g., `/_api/get-tests`), ensure `NODE_ENV` is set to `test` in your environment. On Replit, add a Secret `NODE_ENV=test` or start with `NODE_ENV=test node server.js`.
